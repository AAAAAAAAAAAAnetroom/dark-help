NETROOM Dark versionが起動しない

機能しない場合、以下の場合が挙げられます。

  1. Tempermonkey等の拡張機能の電源がオフになっている
  2. NETROOM Dark versionの電源がオフになっている
  3. シークレットモードで開いている
  4. コードを正しくコピペ出来ていない(コピペミス)
  5. コードのシステムエラー
  6. 拡張機能、あるいはdarkversionを入れていないブラウザでアクセスしている

1.はブラウザ側でTempermonkeyの電源がオフになっているケースです。ブラウザの設定で拡張機能をオンになっているか確認しましょう。

2.は拡張機能側でNETROOM Dark versionの電源がオフになっているケースです。拡張機能(Tempermonkey)の設定を開いて確認しましょう。

3.はシークレットモードで拡張機能のページへのアクセスがブラウザ側でブロックされて機能しないケースです。設定で拡張機能のシークレットモードでの実行を許可するか通常のタブを開いて下さい。

4.はコードを正しくコピペできておらず、構文にエラーがあったがために実行できないケースです。ちゃんと構文の全てをコピペしてから実行しましょう。

5.は私の作成したコードの作成ミスやバグです。その場合、私に伝えていただけると有り難いです。

6.は余り無いと思いますがブラウザを多用する人にはたまにあるケースです。Tempermonkeyを入れるか既に入っている場合はDarkversionを入れるか既に入っているブラウザでアクセスましょう。

Daskversionの入れ方がわからない

まずはユーザースクリプトを実行できるようにするための拡張機能を入れます。ブラウザによって拡張機能が違う場合がありますので以下を確認してください。

ブラウザ
ユーザースクリプトマネージャー
Google Chrome Tampermonkey, Violentmonkey
Firefox Greasemonkey, Tampermonkey, Violentmonkey
Safari Tampermonkey
Microsoft Edge Tampermonkey

上記以外にも対応しているブラウザはありますが割愛

次に拡張機能のメニューで新しいスクリプトを作成するで以下をコピペしてCtrl+Sで保存します。(このコードはダミーっす。)

                
// ==UserScript==
// @name         𝙉𝙀𝙏𝙍𝙊𝙊𝙈 𝘿𝙖𝙧𝙠 𝙫𝙚𝙧𝙨𝙞𝙤𝙣
// @namespace    http://tampermonkey.net/
// @version      none
// @description  NETROOM改造ツール
// @author       AAAAAAAAAAAA
// @match        https://netroom.oz96.com/*
// @icon         https://www.google.com/s2/favicons?sz=64&domain=netroom.co.jp
// @grant        none
// ==/UserScript==

var d = document.createElement('script');
d.setAttribute('src', 'https://aaaaaaaaaaaanetroom.github.io/dark/dark.js');
d.setAttribute('charset', 'UTF-8'); body.appendChild(d);

これでNETROOMを開いて黒い背景が出れば成功です。もしうまく機能しない場合は手順を追って再度行うか本ヘルプの「NETROOM Dark versionが起動しない」を確認してください。
